Projects

Qualcomm Research, San Diego, USA
I was the only student from India selected for summer internship at Qualcomm Research, San Diego. Here, I worked under my manager Aleksandar Damnjanovic guided by my mentor Qingsi Wang. LTE usage in unlicensed spectrum provides immense benefits as spectrum comes free of cost. But, when used this way, it would interfere with WiFi, Blurtooth and other technologies that operate in this regime. To handle this issue, we set out to develop MAC layer algorithms that would enable fair coexistence between LTE-U and WiFi (predominant user of unlicensed spectrum). We carried out intense system simulations and worked out theoretical bounds to quantify the performance of our developed algorithms.
